POST Login
{{base_url}}/v1/auth

This method is used to log in a user. It expects an email and password in the request body. On successful login, it returns an access token in response body and a refresh token in cookies.

BODY raw
                                                  
                                                    
{
    "email": "li@sa.gov.au",
    "password": "member123"
}
Example Request
  • Login
                                                
POST api.sigure.net/v1/auth

{
    "email": "li@sa.gov.au",
    "password": "member123"
}
GET Refresh Token
{{base_url}}/v1/auth

This method is used to refresh the access token of a logged-in user. It expects a refresh token in the cookies. On successful verification of the refresh token, it returns a new access token in response body and a refresh token in cookies.

Example Request
  • Refresh Token
                                                
GET api.sigure.net/v1/auth
DELETE Logout
{{base_url}}/v1/auth

This method is used to log out a user. It expects a refresh token in the cookies. On successful verification of the refresh token, it clears the refresh token and returns a 200 status code.

Example Request
  • Logout
                                                
DELETE api.sigure.net/v1/auth

POST Create a New User
{{base_url}}/v1/users

This method is used to create a new user. It expects a name, email, role, and password in the request body. Only admins can create users. On successful creation, it returns the created user object excluding sensitive information.
